‚è±Ô∏è Complexity Analysis

Time Complexity: O(n)

  • Single pass through n nodes: O(n)
  • Array index access: O(1)
  • Total: O(n) + O(1) = O(n)

Space Complexity: O(n)

  • Array stores n node references: O(n)
  • This is the trade-off for single pass!
‚úÖ Pros:
  • Single traversal
  • Simple logic

‚ùå Cons:
  • O(n) extra memory

  • Not optimal for huge lists
