⚠️ Why This Is Suboptimal
Time Complexity: O(n) - We visit each node twice (once to collect, once to build)
Space Complexity: O(n) - Array of n values + n NEW nodes = 2n extra space!
Problems: Creates entirely new nodes instead of reusing existing ones. The optimal solution uses O(1) space by just changing pointers!
